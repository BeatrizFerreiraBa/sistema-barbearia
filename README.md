# 💈 Sistema Barbearia

Sistema desenvolvido em Java para gerenciamento de uma barbearia, permitindo o cadastro e gerenciamento de clientes, barbeiros, serviços e agendamentos.

> Projeto desenvolvido para praticar Programação Orientada a Objetos (POO), organização de código e boas práticas de desenvolvimento utilizando Java.

## ✨ Funcionalidades

- Cadastro de clientes
- Cadastro de serviços
- Agendamento de atendimentos
- Consulta de registros
